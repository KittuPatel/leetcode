# 1017. Convert to Base -2

## Medium



Given an integer `n`, return _a binary string representing its representation in base_ `-2`.

**Note** that the returned string should not have leading zeros unless the string is `"0"`.

&#x20;

**Example 1:**

<pre><code>Input: n = 2
<strong>Output:
</strong> "110"
<strong>Explantion:
</strong> (-2)2 + (-2)1 = 2
</code></pre>

**Example 2:**

<pre><code>Input: n = 3
<strong>Output:
</strong> "111"
<strong>Explantion:
</strong> (-2)2 + (-2)1 + (-2)0 = 3
</code></pre>

**Example 3:**

<pre><code>Input: n = 4
<strong>Output:
</strong> "100"
<strong>Explantion:
</strong> (-2)2 = 4
</code></pre>

&#x20;

**Constraints:**

* `0 <= n <= 109`
